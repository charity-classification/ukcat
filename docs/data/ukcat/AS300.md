# Youth Groups `AS300`

Youth Groups is a tag in the [UK-CAT classification](/data/tag_list/) of charities in the 
UK. It is part of the [Associations](/data/ukcat/AS) categoryand the Youth Groups subcategory.

1,881 charities have been tagged with this tag.
This represents 0.93% of all charities.

## Keywords to find charities

UK-CAT categories are applied to charities based on a set of keywords, in the form of a [regular expression](https://en.wikipedia.org/wiki/Regular_expression).

The regular expression used to find charities in this tag is:

`\b(youth (group|club)s?)\b`
