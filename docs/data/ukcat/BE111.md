# Racial; ethnic or national communities `BE111`

Racial; ethnic or national communities is a tag in the [UK-CAT classification](../tag_list.md) of charities in the 
UK. It is part of the [Beneficiary group](BE.md) category.

625 charities have been tagged with this tag.
This represents 0.31% of all charities.

## Keywords to find charities

UK-CAT categories are applied to charities based on a set of keywords, in the form of a [regular expression](https://en.wikipedia.org/wiki/Regular_expression).

The regular expression used to find charities in this tag is:

`\b(B\.?(A\.?)?M\.?E|racism|ethnic minorit(ies|y)|multi[\W ]+ethnic|black british)\b`
