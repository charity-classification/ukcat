# Women `BE113`

Women is a tag in the [UK-CAT classification](/data/tag_list/) of charities in the 
UK. It is part of the [Beneficiary group](/data/ukcat/BE) category.

9,724 charities have been tagged with this tag.
This represents 4.83% of all charities.

## Keywords to find charities

UK-CAT categories are applied to charities based on a set of keywords, in the form of a [regular expression](https://en.wikipedia.org/wiki/Regular_expression).

The regular expression used to find charities in this tag is:

`\b(wom[ae]n'?s?|widows?|girl'?s?|mum'?s?|ladies|females?|abort\w*|natal|pregnancy|maternity|sister(s)?|spinsters?)\b`


Charities matching this regular expression will be excluded from the tag:

`\b(m[ae]n and|and m[ae]n|boys and|and boys)\b`
