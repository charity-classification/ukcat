# Advice and individual advocacy `CA101`

Advice and individual advocacy is a tag in the [UK-CAT classification](/data/tag_list/) of charities in the 
UK. It is part of the [Charitable activities](/data/ukcat/CA) category.

3,248 charities have been tagged with this tag.
This represents 1.61% of all charities.

## Keywords to find charities

UK-CAT categories are applied to charities based on a set of keywords, in the form of a [regular expression](https://en.wikipedia.org/wiki/Regular_expression).

The regular expression used to find charities in this tag is:

`\b(information service|law centre|law society|legal (rights|services?|advocacy|support|advice)|(debt|financial) advice|advocacy|help ?-?line|CITIZENS ADVICE|advice (&|and) guidance)\b`
