# Crime and Justice `CJ`

Crime and Justice is a tag in the [UK-CAT classification](/data/tag_list/) of charities in the 
UK. It is part of the [Crime and Justice](/data/ukcat/CJ) category.

1,831 charities have been tagged with this tag.
This represents 0.91% of all charities.

## Keywords to find charities

UK-CAT categories are applied to charities based on a set of keywords, in the form of a [regular expression](https://en.wikipedia.org/wiki/Regular_expression).

The regular expression used to find charities in this tag is:

`\b(crime|criminal)\b`
