# School fundraising `ED202`

School fundraising is a tag in the [UK-CAT classification](/data/tag_list/) of charities in the 
UK. It is part of the [Education](/data/ukcat/ED) categoryand the School support subcategory.

4,962 charities have been tagged with this tag.
This represents 2.46% of all charities.

## Keywords to find charities

UK-CAT categories are applied to charities based on a set of keywords, in the form of a [regular expression](https://en.wikipedia.org/wiki/Regular_expression).

The regular expression used to find charities in this tag is:

`\b(school (fund|foundation|association)|benefit of (the )?pupils|(funds|fundraising|donations) (for|to) .* (school|pupils))\b`
