# Employability training `ED302`

Employability training is a tag in the [UK-CAT classification](/data/tag_list/) of charities in the 
UK. It is part of the [Education](/data/ukcat/ED) categoryand the Training subcategory.

4,450 charities have been tagged with this tag.
This represents 2.21% of all charities.

## Keywords to find charities

UK-CAT categories are applied to charities based on a set of keywords, in the form of a [regular expression](https://en.wikipedia.org/wiki/Regular_expression).

The regular expression used to find charities in this tag is:

`\b(work ?-?(support|preparation|placements?|experience)|employ(ment|ability)|unemploy(ed|ment)|entrepreneur(ial)?|activation programmes?|(labour|jobs?) market)\b`
