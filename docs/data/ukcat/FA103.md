# Community centre `FA103`

Community centre is a tag in the [UK-CAT classification](../tag_list.md) of charities in the 
UK. It is part of the [Facilities](FA.md) category.

3,444 charities have been tagged with this tag.
This represents 1.70% of all charities.

## Keywords to find charities

UK-CAT categories are applied to charities based on a set of keywords, in the form of a [regular expression](https://en.wikipedia.org/wiki/Regular_expression).

The regular expression used to find charities in this tag is:

`\b(community( and)?( sports?)? cent(re|er)s?|community action)\b`
