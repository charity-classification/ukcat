# Accommodation `HO101`

Accommodation is a tag in the [UK-CAT classification](/data/tag_list/) of charities in the 
UK. It is part of the [Housing](/data/ukcat/HO) category.

8,225 charities have been tagged with this tag.
This represents 4.06% of all charities.

## Keywords to find charities

UK-CAT categories are applied to charities based on a set of keywords, in the form of a [regular expression](https://en.wikipedia.org/wiki/Regular_expression).

The regular expression used to find charities in this tag is:

`\b(hous(ing|es)|land(lord|lady)|home(s)?)+\b`
