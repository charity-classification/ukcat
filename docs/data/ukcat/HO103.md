# Homelessness `HO103`

Homelessness is a tag in the [UK-CAT classification](../tag_list.md) of charities in the 
UK. It is part of the [Housing](HO.md) category.

2,417 charities have been tagged with this tag.
This represents 1.19% of all charities.

## Keywords to find charities

UK-CAT categories are applied to charities based on a set of keywords, in the form of a [regular expression](https://en.wikipedia.org/wiki/Regular_expression).

The regular expression used to find charities in this tag is:

`\b(homeless(ness)?|rough( |-)?sleep(ers?|ing)|no fixed abode|on the streets?)\b`
