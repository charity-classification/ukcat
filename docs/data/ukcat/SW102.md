# Bereavement `SW102`

Bereavement is a tag in the [UK-CAT classification](../tag_list.md) of charities in the 
UK. It is part of the [Social welfare](SW.md) category.

1,071 charities have been tagged with this tag.
This represents 0.53% of all charities.

## Keywords to find charities

UK-CAT categories are applied to charities based on a set of keywords, in the form of a [regular expression](https://en.wikipedia.org/wiki/Regular_expression).

The regular expression used to find charities in this tag is:

`\b(bereave(ment|d)|grief|grieving|still-? ?birth|(loss of|lost) a (child|parent|baby|loved one|spouse|partner))\b`


Charities matching this regular expression will be excluded from the tag:

`(sick and (the )?bereaved)`
