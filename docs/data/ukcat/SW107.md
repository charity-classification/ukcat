# Social activities `SW107`

Social activities is a tag in the [UK-CAT classification](/data/tag_list/) of charities in the 
UK. It is part of the [Social welfare](/data/ukcat/SW) category.

5,385 charities have been tagged with this tag.
This represents 2.66% of all charities.

## Keywords to find charities

UK-CAT categories are applied to charities based on a set of keywords, in the form of a [regular expression](https://en.wikipedia.org/wiki/Regular_expression).

The regular expression used to find charities in this tag is:

`\b(social (activities|interaction|events?|inclusion)|sociali[zs]ing|sociali[sz]e|companionship|friendship)\b`
